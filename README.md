# 1️⃣ What is the difference between var, let, and const?
- var is functional-scoped. It can be redeclared and reassigned and also hoisted.
- let is block-scoped. It can be reassigned but cannot be redeclared.
- const is also block-scoped. But it cannot be reassigned or redeclared.


# 2️⃣ What is the spread operator (...)?
<p>The spread operator (...) in JavaScript is used to expand an iterable into individual elements like Array.</p>


# 3️⃣ What is the difference between map(), filter(), and forEach()?
- map() : this method returns transformed array
- filter() : this method returns filtered array
- forEach() : this method just loops, no return


# 4️⃣ What is an arrow function?
<p>An arrow function is a syntax for writing JavaScript function expressions using an "arrow" (=>) and it is introduced in ES6.</p>


# 5️⃣ What are template literals?
<p>Template literals are a way to create strings in JavaScript using backticks (``) instead of quotes. It allows interpolation, multi-line strings, and expressions.</p>
